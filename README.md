# 抖喵游戏屋

一个使用 React + TypeScript + Node.js + Socket.IO 搭建的实时联机小游戏项目。当前主玩法是锤子剪刀布，包含大厅、房间、聊天、观战、Bot、排位、惩罚模式、名字争夺战、后台文本配置工具等功能。

## 本地运行

```bash
npm install
npm run dev
```

- 前端默认地址：`http://127.0.0.1:5173`
- 后端默认地址：`http://127.0.0.1:3000`

## 构建检查

每次修改后建议运行：

```bash
npm run build
npm run server:build
```

## 后台与配置

- 后台入口：`/admin` 或 `#admin`
- 仓库默认不保存后台口令。
- 部署时建议用环境变量设置后台口令：

```bash
ADMIN_PASSWORD=你的后台口令
```

配置文件说明：

- `config/default.json`：默认配置，会提交到 Git。
- `config/active.json`：服务器运行时实际配置，已被 `.gitignore` 忽略，不会提交。
- `work/uploads`：玩家证明图片和后台图片上传目录，已被 `.gitignore` 忽略。
